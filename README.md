# threejs-activity3-3dmaze-
A 3D Maze Game built using **Three.js** and **Electron**, featuring:
- Physics-based ball movement
- Simple AABB collision
- Multi-level JSON maps
- Confetti particle win effect
- Joystick or keyboard control

## How to Run

1. Install dependencies:
   ```bash
   npm install

2. Start the game
   ```bash
   npm start
